Cliché Magazine

Some pages I formatted in a pleasing format to publish bimonthly magazine issues that has 4,500,000 readers yearly.

challenge
Assist in developing the layout of articles and incorporating ideas from the inspiration of layouts collected by the Editor in Chief. With the brand identity guidelines, the Editor in Chief wanted creative layouts.

solution
Listened attentively and communicated with the Editor in Chief on each issue as we worked with tight deadlines and brand guidelines. I looked at Cliches previous magazine issues to take note how the layouts looked. Then I would check the Editor in Chief’s folder of inspiration to see what can be incorporated if it calls for it.
